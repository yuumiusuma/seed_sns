物理削除 論理削除

物理削除...実際にデータをテーブルから削除する方法
SQL ... DELETE文を使用
メリット : 不要なデータを貯めておかないため、容量の節約になる。

論理削除...フラグ制御によって、一覧からデータを非表示にし削除されたように見せる方法
SQL ... UPDATE文を使用
メリット : データの復元が可能。重要なデータを物理削除した場合、手軽に復元することができないため、ユーザー情報等の大切なデータは実際のシステムでは論理削除されていることが多い

テーブルのユーザー定義について
テーブルのカラムへ挿入するデータが未指定の時、セットする値を決められる。(INSERT文に設定しなくてもエラーにはならない)

論理削除の例
tweet_idが1のものを論理削除する

UPDATE `tweets` SET `delete_flag`=1 WHERE `tweet_id`=1;
